# Proyecto Tetris - Prototipos HTML

## Descripción

Este repositorio contiene **prototipos de juegos en HTML** desarrollados como prueba de concepto para validar mecánicas de juego antes de crear la versión final en Unity.

## Estructura

```
Tetris/
├── index.html              # Menú principal con enlaces a los juegos
├── index_chatgpt.html      # Tetris generado por ChatGPT
├── index_claude.html       # Tetris generado por Claude
├── index_gemini.html       # Tetris generado por Gemini
├── Defensoresdegamonal/     # Prototipo de juego "Defensores de Gamonal"
│   ├── especificacion.md
│   ├── defensores_gemini.html
│   └── defensores-claude.html
└── TresEnRaya/             # Prototipo de Tres en Raya
    ├── index_minimax.html
    └── index_claude.html
```

## Propósito

Cada archivo HTML es un juego funcional creado por diferentes modelos de IA para:
- Validar mecánicas de juego
- Iterar rápidamente en el diseño
- Servir como referencia para la implementación en Unity

## Uso

Abrir `index.html` en un navegador para acceder a los diferentes prototipos.
